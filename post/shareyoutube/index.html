没有VPN也可以看Youtube？！

2020-05-11 youtube VPN GitHub

今天无意中发现一个神奇的网站，可以没vpn就能在线看油管。但是设计者的初心好像是分享你的油管。比如自己看到有趣的油管视频想分享给朋友但是朋友没有vpn这些的高端操作；去教他反而他因为怕麻烦不感兴趣，毁了当初分享快乐的好心情。而这个网站刚刚好可以帮助到你！但是不要用它来疯狂看油管视频哦～白嫖要温柔～🤠

网址:https://s.fqjh.cf

超具体的使用方法：

1.首先打开自己的Youtube点击分享按钮（处在vpn环境）

查看图片详解 ![点击分享](https://cdn.jsdelivr.net/gh/closty/tuchuang/usr/uploads/2020/05/465202753.jpg)

2.复制分享链接

查看图片详解 ![复制分享链接](https://cdn.jsdelivr.net/gh/closty/tuchuang/usr/uploads/2020/05/3757569177.jpg)

3.将复制好的分享链接填入到https://s.fqjh.cf 的框框中，在点击submit

查看图片详解 ![FDC047F6EDA7396E5865232CCD681D56.jpg](https://cdn.jsdelivr.net/gh/closty/tuchuang/usr/uploads/2020/05/1581818605.jpg)

4.缓慢的转到这个网址，点击sharelink的链接就能直接复制啦！

查看图片详解 ![D181B25FE570DA4E043A3045C71F0CFD.jpg](https://cdn.jsdelivr.net/gh/closty/tuchuang/usr/uploads/2020/05/1626273878.jpg)

5.复制的sharelink便可以无vpn的情况下在线网页看YouTube了！你可以将你喜欢的YouTube视频分享给你的朋友而不用自己麻烦的下载转发亦或是教朋友如何使用代理了（相信我他会嫌你麻烦）
